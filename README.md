# Secure Banking Simulation System

A full-stack banking application developed using Spring Boot, React JS, and MySQL. The project simulates core banking operations such as account creation, deposit, withdrawal, money transfer, transaction history management, and secure user authentication.

---

## Features

- User Authentication Login System
- Protected Dashboard Access
- Create Bank Account
- Deposit Money
- Withdraw Money
- Transfer Money Between Accounts
- Delete Bank Account
- Transaction History Tracking
- REST API Integration
- MySQL Database Integration

---

## Technologies Used

### Backend
- Java
- Spring Boot
- Spring Security
- Hibernate JPA
- Maven

### Frontend
- React JS
- Axios
- HTML
- CSS

### Database
- MySQL

---

## System Architecture

User → React Frontend → REST APIs → Spring Boot Backend → MySQL Database

---

## Modules

### Authentication Module
Handles user login and protected access.

### Account Management Module
Handles account creation and deletion.

### Transaction Module
Handles deposit, withdrawal, transfer, and transaction history.
